# Face Mask Detection System

## Description
This project is a Face Mask Detection System developed using Python, Flask, OpenCV, and YOLOv8. Users can upload an image, and the application detects objects in the image using the YOLOv8 model.

## Features
- Upload an image
- Object detection using YOLOv8
- Display detected objects
- Web interface using Flask

## Technologies Used
- Python
- Flask
- OpenCV
- Ultralytics YOLOv8
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open:

```
http://127.0.0.1:5005
```
