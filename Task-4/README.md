# AI Resume Screening System

## Description
This project is an AI-based Resume Screening System developed using Python and Flask. It allows users to upload a PDF resume, extracts the resume text, compares it against a predefined set of skills, and displays the matching percentage along with matched and missing skills.

## Features
- Upload PDF Resume
- Extract Resume Text
- Skill Matching
- Resume Score Calculation
- Displays Matched Skills
- Displays Missing Skills
- Simple Flask Web Interface

## Technologies Used
- Python
- Flask
- PyMuPDF
- HTML
- CSS
