# AI-Based Sentiment Analysis for Social Media

## Description
This project analyzes social media text and classifies it as **Positive**, **Negative**, or **Neutral** using Natural Language Processing (NLP) and Machine Learning.

## Features
- Positive Sentiment Detection
- Negative Sentiment Detection
- Neutral Sentiment Detection
- TF-IDF Text Vectorization
- Naive Bayes Classifier
- Flask Web Interface

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- Joblib
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
python train_model.py
python app.py
```

http://127.0.0.1:5008
