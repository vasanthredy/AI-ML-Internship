# AI-Based Handwritten Digit Recognition

## Description
This project recognizes handwritten digits (0–9) using Machine Learning with Scikit-learn.

## Features
- Predict handwritten digits
- Random Forest Classifier
- Flask Web Interface
- Simple and easy to use

## Technologies Used
- Python
- Flask
- Scikit-learn
- NumPy
- Joblib
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
python train_model.py
python app.py
```

Open:

```
http://127.0.0.1:5007
```
